fswatch

Documentation

fswatch provides the following documentation:

  • Texinfo documentation, included with the distribution.
  • HTML documentation (generated with Texinfo).
  • PDF documentation (generated with Texinfo).
  • Doxygen documentation of the libfswatch API.
  • A wiki page.
  • A man page.

fswatch official documentation is provided in Texinfo format. This is the most comprehensive source of information about fswatch. The man page, in particular, is a stub that suggests the user to use the info page instead.

If you are installing fswatch using a package manager and you would like the PDF manual to be bundled into the package, please send a feature request to the package maintainer.
